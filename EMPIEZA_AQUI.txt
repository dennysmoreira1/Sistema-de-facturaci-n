⚠️  IMPORTANTE - Crear archivo .env:

   Crea un archivo llamado ".env" en la raíz con:

   DATABASE_URL="file:./dev.db"
   NEXTAUTH_URL="http://localhost:3000"
   NEXTAUTH_SECRET="cambia-por-clave-segura"

   (Puedes copiar .env.example y renombrarlo a .env)


📁 ESTRUCTURA DEL PROYECTO:

   app/
   ├── (auth)/           → Login y Registro
   ├── (dashboard)/      → Dashboard, Clientes, Facturas
   └── api/              → API Routes

   components/
   ├── ui/               → Componentes reutilizables
   └── layout/           → Sidebar, Navbar

   lib/
   ├── prisma.ts         → Base de datos
   ├── auth.ts           → Autenticación
   └── exportPDF.ts      → Exportar PDFs

   prisma/
   ├── schema.prisma     → Modelos de datos
   └── seed.ts           → Datos de prueba


✨ FUNCIONALIDADES IMPLEMENTADAS:

   ✅ Autenticación completa (Login/Registro/Logout)
   ✅ Dashboard con KPIs en tiempo real
   ✅ CRUD de Clientes (Crear/Editar/Eliminar)
   ✅ CRUD de Facturas con items dinámicos
   ✅ Cálculos automáticos de totales
   ✅ Exportar facturas a PDF
   ✅ Exportar listado a CSV
   ✅ Filtros por estado y cliente
   ✅ Búsqueda de clientes
   ✅ Diseño moderno y responsive
   ✅ Animaciones personalizadas
   ✅ 100% TypeScript


🛠️  TECNOLOGÍAS USADAS:

   • Next.js 14 (App Router)
   • TypeScript
   • Tailwind CSS
   • Prisma ORM
   • SQLite / PostgreSQL
   • NextAuth.js
   • jsPDF
   • Lucide Icons


🚀 COMANDOS ÚTILES:
